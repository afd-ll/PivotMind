# 玄枢 (PivotMind) — 溯智系统

**纯C实现的认知AI框架**，基于多拓扑网络的自主学习和推理引擎。

47个源文件，约5.4万行C代码，全部手写。在树莓派3B上7×24运行。

## 架构概览

```
输入 → 语义理解 → 多拓扑联想推理 → Seq2Seq神经网络 → 回复
         ↓                                      ↑
     auto_learn_concepts() ← 在线训练 ←──────────┘
         ↓
     记忆系统 + 概念层次结构
         ↓
     主动学习器 (后台每5分钟)
         ├─ 节点重要性评估 (PageRank)
         ├─ 共现关系发现
         ├─ 拓扑自动增长/收缩
         ├─ 遗忘机制 (低置信度清理)
         └─ EWC持续学习巩固
```

## 核心组件

| 模块 | 功能 | 行数 |
|------|------|------|
| 多拓扑网络 | 8层概念拓扑（词汇/语义/情绪/语法/上下文/领域/语用/文化） | ~1500 |
| 华容道拓扑 | 推理节点 + 联想传播 + 激活扩散 | ~1500 |
| 对话系统 | 意图识别 + 实体提取 + 拓扑驱动回复生成 | ~1700 |
| 主动学习器 | 后台线程持续学习 + 关系发现 + 拓扑增长 | ~800 |
| Seq2Seq模型 | Encoder-Decoder RNN + BPTT在线训练 | ~400 |
| 概念抽象 | 7层概念层次（具体→感觉→空间→时间→分类→因果→意图） | ~400 |
| 因果推理 | 因果关系图 + 因果路径推理 | ~2000 |
| 记忆系统 | 三级记忆（上下文/短期/永久） | ~850 |
| 神经网络层 | Embedding / RNN / LSTM / GRU / Linear / Attention | ~3000 |
| 中文分词 | UTF-8中文分词器 | ~100 |
| 概念处理 | 数学表达式计算 + 规则学习 | ~270 |
| 持久化 | 拓扑状态保存/加载 | ~150 |

## 快速开始

```bash
# 编译
make linux

# 运行
./build/bin/digital_life

# 对话（内置主动学习，持续成长）
# 输入 'help' 查看命令
# 输入 'quit' 退出
```

## 构建选项

| 命令 | 说明 |
|------|------|
| `make linux` | 全量编译（先清理再构建） |
| `make lib` | 仅构建静态库 libpivotmind.a |
| `make run` | 构建并运行 |
| `make clean` | 清理构建产物 |

## 依赖

- GCC/Clang (C99 + GNU扩展)
- pthread (多线程)
- SQLite3 (词典支持)

## 技术特点

- **全C实现** — 无Python/无第三方AI框架依赖
- **在线学习** — 每轮对话都是训练样本，无需预训练
- **持续运行** — 设计为7×24小时后台服务
- **低资源** — 树莓派3B (1GB RAM) 可流畅运行
- **拓扑驱动** — 概念网络自适应增长/遗忘
- **多维度** — 词汇/语义/情绪/语法/上下文多层次推理
