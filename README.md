# 玄枢 (PivotMind) — 溯智系统

**纯C实现的认知AI框架**，基于多拓扑网络的自主学习和推理引擎。

47个源文件，约5.4万行C代码，全部手写，无第三方AI框架依赖。

## 重要说明

> ⚠️ **这不是开箱即用的聊天机器人。**
>
> 这是一个**认知架构研究框架**，核心机制是多拓扑网络+在线持续学习。
> 对话能力是从零开始、通过交互逐步成长的——刚开始只会说"我想，联想到【X】、【Y】。"
> 聊得越多，句型和知识越丰富。系统设计为7×24小时运行，与它对话就是训练它。

## 支持平台

| 平台 | 状态 | 说明 |
|------|------|------|
| Linux | ✅ 支持 | 主目标平台，树莓派3B实测通过 |
| ARM Linux (树莓派) | ✅ 支持 | 905Mi内存 + 4核Cortex-A53 实测运行 |
| x86 Linux | ✅ 支持 | 标准GCC编译 |
| macOS | ⚠️ 未测试 | 理论上可编译，需验证 |
| Windows (WSL) | ⚠️ 需WSL | 原生Win32支持已移除(遗留代码已清理) |
| Windows (原生) | ❌ 不支持 | 旧Windows代码已删除 |

**硬件限制：**
- 最低内存: 512MB（编译需~1.5GB，建议使用swap）
- 磁盘: 编译产物约500KB，运行时约50MB（拓扑数据）
- 主动学习器后台线程会定期执行计算密集型操作（PageRank、EWC等），低功耗设备上可能影响响应速度

## 架构概览

```mermaid
graph TD
    subgraph 自主层【运行时自动】
        A[输入] --> B[激活概念节点]

    end

    subgraph 非自主层【交互时触发】

    F -.->|不满意| J

    M -.->|置信度更新| D
```

## 快速开始

```bash
# 编译（Linux/树莓派）
make linux

# 运行
./build/bin/digital_life

# 命令
#   help    - 显示帮助
#   quit    - 退出
#   stats   - 学习统计
#   network - 查看拓扑网络
#   memory  - 查看记忆状态
#   learn   - 手动触发学习周期

# 直接对话即可，每次对话都会自动学习
```

## 构建选项

| 命令 | 说明 |
|------|------|
| `make linux` | 全量编译（先清理再构建，约3分钟在树莓派3B上） |

## 技术特点

- **全C实现** — 无Python/无第三方AI框架依赖，纯手写
- **在线持续学习** — 无需预训练/无需数据集，每轮对话即训练样本
- **概念网络自增长** — 拓扑节点随对话自动扩展（从14个初始节点起步）
- **后台主动学习** — 独立线程每5分钟执行概念巩固/关系发现/遗忘
- **低资源运行** — 树莓派3B (1GB RAM, 4核1.2GHz) 可流畅运行
- **多维度推理** — 词汇/语义/情绪/语法/上下文/领域/语用/文化八层拓扑同时激活
- **状态持久化** — 拓扑网络可保存到文件，重启后恢复（约50KB/2000节点）
- **三权分立架构** — 拓扑网络(联想) + 记忆系统(存储) + 主动学习器(成长) 独立运行
- **走边路径生成** — 从高激活节点出发沿边贪心行走，六维混合评分选下一步
- **认知调度中心** — 意图向量引导搜索方向，retry循环确保输出质量
